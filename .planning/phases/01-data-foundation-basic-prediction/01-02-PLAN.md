    - `generate_lag_features(df: pd.DataFrame, lag_hours: list[int] = [24, 168]) -> pd.DataFrame`（Tier 2）：
      - 对 `lag_hours` 中的每个 `lag`：创建列 `lag_{lag}h` = `df['load_mw'].shift(lag)`。
      - 用 `fillna(method='bfill')` 填充 shift 产生的前导 NaN。
      - 返回 lag 特征列的 DataFrame。

    - `generate_rolling_features(df: pd.DataFrame, windows: list[int] = [24]) -> pd.DataFrame`（Tier 3）：
      - 对 `windows` 中的每个 `w`：
        - `rolling_mean_{w}h` = `df['load_mw'].rolling(window=w, min_periods=1).mean()`

      - 填充 rolling 特征中的初始 NaN（窗口未满时）使用扩展窗口等效值（已由 min_periods=1 处理）。
      - 返回 rolling 特征列的 DataFrame。

    - `generate_all_features(df: pd.DataFrame, tier: int = 3) -> pd.DataFrame`：
      - Tier 1：`calendar = self.generate_calendar_features(df)`。返回 `calendar` 与原始 `timestamp` + `load_mw` 的连接。
      - Tier 2：calendar + `lag = self.generate_lag_features(df)`。返回 `pd.concat([calendar, lag], axis=1)` 含 timestamp+load_mw。
      - Tier 3：calendar + lag + `rolling = self.generate_rolling_features(df)`。返回 `pd.concat([calendar, lag, rolling], axis=1)` 含 timestamp+load_mw。
      - 在输出 DataFrame 中将 `timestamp` 和 `load_mw` 作为前两列，方便使用。
      - 注意：特征以未缩放形式返回（按 D-09 缩放由预测器负责）。

    - `get_feature_names(self, tier: int = 3) -> list[str]`：
      - 返回给定 tier 的特征列名有序列表（不包括 timestamp 和 load_mw）。

    **Pitfall 1/7 强制执行说明**：FeatureEngineer 不 fit scaler、不选择特征、不在完整数据集上计算统计量。它仅创建确定性的 lag/rolling/calendar 列。预测器在每个 TimeSeriesSplit 折内处理缩放。
