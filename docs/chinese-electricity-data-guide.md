# 中国电力公开数据获取指南

> 本文档指导如何手动获取中国电力负荷/市场数据，用于 Ellectric 学习项目。
> 所有数据源均为公开可获取，不涉及商业数据购买。

## 0. 背景：为什么需要手动获取？

经测试，中国电力数据平台（国家统计局、地方开放数据平台、中电联、天池、和鲸等）均设置了反爬机制（验证码、JS渲染、TLS限制），无法通过Python程序自动获取。因此 Phase 1 采用**手动下载 + 本地文件加载**的方式。

---

## 1. 优先推荐：和鲸社区 (HeyWhale) 数据集

**地址：** https://www.heywhale.com/datasets

**为什么推荐：** 数据科学社区，大量用户上传的公开数据集，很多是CSV/Excel格式，直接下载即可。

**搜索关键词：**
- "电力负荷"、"电力数据"、"用电量"
- "电力市场"、"电价"
- "能源"、"发电"
- 具体省份：广东电力、江苏电力、浙江电力

**示例数据集（搜索尝试）：**
1. 搜索 "电力负荷 预测" → 通常有带时间序列的负荷数据
2. 搜索 "电力市场 出清" → 可能有电价和交易数据
3. 搜索具体年份如 "2019 电力 数据"

**下载步骤：**
1. 注册账号（微信/手机号）
2. 搜索上述关键词
3. 找到包含 "CSV" 或 "Excel" 格式的数据集
4. 点击 "下载" 按钮
5. 保存到项目的 `data/` 目录

## 2. 阿里云天池数据集

**地址：** https://tianchi.aliyun.com/dataset

**搜索关键词：** "电力负荷"、"电力预测"、"能耗"、"用电"

**特点：** 阿里云平台，数据集通常来源于竞赛或研究，质量较高。可能需要阿里云账号。

**推荐数据集：**
- 搜索 "电力AI大赛" 或 "电力预测" — 天池举办过多次电力预测竞赛，附带优质训练数据
- "扬中市高新园区用电" / "工业用电预测" — 实际工业用电数据

## 3. 地方公共数据开放平台（手动浏览器访问）

这些平台需要**真实浏览器**访问，通常有验证码。

| 平台 | 地址 | 数据特点 |
|------|------|---------|
| 山东省公共数据 | http://data.sd.gov.cn/ | 电力、能源相关数据集 |
| 广东省公共数据 | https://data.gd.gov.cn/ | 省级能源数据 |
| 浙江省公共数据 | https://data.zj.gov.cn/ | 电力消费数据 |
| 深圳市开放数据 | https://opendata.sz.gov.cn/ | 城市级用电数据 |
| 上海市开放数据 | https://data.sh.gov.cn/ | 能源消耗数据 |
| 北京市开放数据 | https://data.beijing.gov.cn/ | 市政能源数据 |

**操作步骤：**
1. 用 Chrome/Edge 浏览器打开上述网站
2. 搜索 "电力"、"负荷"、"用电量"、"发电"
3. 查看数据格式（CSV/Excel/JSON），筛选小时级或日级数据
4. 下载到本地
5. 放入项目 `data/` 目录

## 4. 国家能源局 (NEA) 统计数据

**地址：** https://www.nea.gov.cn/ → "统计数据" 栏目

**数据内容：**
- 全国电力工业统计数据（月度）
- 风光装机容量、发电量
- 全社会用电量

**格式：** 通常是 HTML 表格或 PDF，需要手动提取到 CSV。

**适合场景：** 宏观趋势分析，不适合直接做小时级负荷预测。可作为补充数据，理解中国电力结构。

---

## 5. 中国电力企业联合会 (CEC)

**地址：** https://www.cec.org.cn/

**数据内容：**
- 电力供需分析报告
- 电力市场交易信息
- 年度/季度电力数据

**获取方式：** 部分报告免费下载，部分需要注册。格式为 PDF/Excel。

---

## 6. EPS 数据平台（学术向）

**地址：** https://www.epsnet.com.cn/

**特点：** 学术级数据库，覆盖中国电力全维度数据。多数高校已购买。

**如果你在学校：**
1. 通过学校图书馆网站 → 电子资源 → EPS 数据平台
2. 搜索 "电力" 相关指标
3. 可导出为 Excel/CSV

## 7. 备用方案：国际数据 + 对比学习

如果短期无法获取中国数据，项目已内置 fallback：

```python
# data_loader.py
from ellectric.pipeline.data_loader import EpftoolboxLoader, ChineseDataLoader

# 方式1：先用国际数据跑通
loader = EpftoolboxLoader(market='PJM')  # 美国PJM市场
df = loader.load_hourly_demand()

# 方式2：获取到中国数据后一键切换
loader = ChineseDataLoader(data_path='data/china_load_2023.csv')
df = loader.load_hourly_demand()
```

## 8. 期望的数据格式

获取到数据后，整理为以下格式之一即可直接用：

**CSV 格式：**
```csv
timestamp,load_mw,region
2023-01-01 00:00:00,45280.5,广东
2023-01-01 01:00:00,42890.3,广东
...
```

**Excel 格式同理，确保包含：**
- 时间列（任何标准格式均可，pandas 会自动解析）
- 负荷值列（单位可以是 MW/万kW/亿kWh，代码会做单位转换）
- 可选：区域列、价格列
