    **pipeline/forecaster.py** — 按 D-12、D-13 实现持久性基线 + 最小 P&L：

    函数 `persistence_forecast(df: pd.DataFrame, shift_hours: int = 24) -> pd.Series`：
    - 将 `load_mw` 列偏移 `shift_hours` 个周期：`df['load_mw'].shift(shift_hours)`。
    - 用 `fillna(method='bfill')` 填充前导 NaN 值（前 `shift_hours` 行）。
    - 返回与输入相同索引的预测 Series。

    函数 `calculate_pnl(actual: pd.Series, forecast: pd.Series, price_per_mwh: float = 50.0) -> pd.DataFrame`：
    - `trade_pnl = (forecast - actual) * price_per_mwh` — 预测偏高时为正（以较低预测价格买入少于实际所需），预测偏低时为负。
    - `cumulative_pnl = trade_pnl.cumsum()`。
    - 返回含列 `[timestamp, actual_mw, forecast_mw, trade_pnl, cumulative_pnl]` 的 DataFrame。timestamp 使用 actual Series 的 index（如果有），否则使用 range index。

    函数 `plot_pnl(pnl_df: pd.DataFrame) -> go.Figure`：
    - 使用 `plotly.subplots.make_subplots(rows=2, cols=1, shared_xaxes=True, vertical_spacing=0.08, subplot_titles=('Actual vs Persistence Forecast', 'Cumulative P&L'))`。
    - 子图 1（上）：`go.Scatter` 绘制实际负荷（`actual_mw`，line=blue，name='Actual'）和预测负荷（`forecast_mw`，line=orange，name='Persistence Forecast'）。
