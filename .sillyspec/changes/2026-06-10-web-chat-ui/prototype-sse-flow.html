⚡ Ellectric Chat — SSE 流式对话数据流

USER 输入: "帮我仿真夏季高峰7天"
POST fetch('/chat/stream', { query, history })
API server.py GET /chat/stream → create_agent_executor() → astream_events()
SSE event: status → data: {"type":"status","content":"thinking"}
前端: 显示"思考中..."
TOOL event: tool_call → data: {"type":"tool_call","tool":"run_simulation"}
前端: 显示黄色工具状态标签 + spinner
SSE Agent 调用 POST /simulate → pipeline → 返回结果
TOOL event: tool_result → data: {"type":"tool_result","tool":"run_simulation"}
前端: 标签变绿色 "run_simulation · 完成"
SSE event: token → data: {"type":"token","content":"夏季"}
SSE event: token → data: {"type":"token","content":"高峰仿真"}
SSE event: token → data: {"type":"token","content":"结果：..."}
...更多 token... 前端逐字渲染 + Markdown + 光标闪烁
SSE event: done → 数据: {"type":"done"}
前端: 移除光标，最终 Markdown 渲染
